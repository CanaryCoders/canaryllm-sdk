# @canarycoders/canaryllm — DEPRECATED

Renamed to **[@canarycoders/ai](https://www.npmjs.com/package/@canarycoders/ai)**.

```bash
npm uninstall @canarycoders/canaryllm && npm install @canarycoders/ai
```

The `0.2.x` line is a thin re-export shim forwarding everything to `@canarycoders/ai`, so existing imports keep working — update them when convenient. The default API base URL is now `https://api.ai.canarycoders.es` (the old `canaryllm.canarycoders.es` still works).
